## Project Description

- This is a personal portfolio website developed to present my academic background, technical skills, projects, and professional profile in one place.
- The main purpose of this portfolio is to support internship applications, job opportunities, and personal branding as an IT undergraduate.
- The website includes important sections such as Home, About Me, Education, Skills, Projects, Resume, and Contact.
- It highlights my software development projects with project images, short descriptions, and relevant details.
- The portfolio is designed with a modern, clean, and responsive user interface for both desktop and mobile users.
- HTML, CSS, JavaScript, and Tailwind CSS are used to build the frontend design and interactions.
- The project is maintained inside a Next.js project structure, with portfolio assets stored in the `public/portfolio` directory.
- Resume PDF files, profile images, logos, and project screenshots are included as supporting resources.
- This portfolio helps visitors quickly understand my skills, experience, and the type of work I can do.
- It also provides an easy way for recruiters, lecturers, or collaborators to contact me.
